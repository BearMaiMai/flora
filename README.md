# 🌸 养花呀 (Flora)

> 一款面向花卉种植爱好者的微信小程序

## 📖 项目简介

**养花呀** 是一款帮助用户科学养花的微信小程序，提供花卉百科查询、智能种植提醒、成长日记记录、拍照识花等功能，让每一位花卉爱好者都能轻松享受种植的乐趣。

## ✨ 核心功能

| 功能模块 | 说明 |
|---------|------|
| 🌺 花卉百科 | 50+ 种常见花卉的养护指南，涵盖浇水、光照、施肥、换盆等 |
| ⏰ 智能提醒 | 基于花卉特性自动生成浇水、施肥提醒，支持订阅消息推送 |
| 📔 成长日记 | 记录植物成长过程，支持图文打卡，生成时间线 |
| 📷 拍照识花 | AI 图像识别，拍照即可识别花卉品种并获取养护建议 |
| 👤 我的花园 | 管理个人种植的花卉，一目了然查看所有植物状态 |
| 🌍 社区分享 | 分享种植心得，与花友交流互动（v2.0 规划） |

## 🛠️ 技术栈

- **前端**：微信小程序原生开发（WXML + WXSS + JavaScript）
- **后端**：微信云开发（CloudBase）
  - 云函数（Node.js）
  - 云数据库（MongoDB）
  - 云存储（文件管理）
- **AI 能力**：百度 AI 植物识别 API

## 🚀 快速开始

### 环境准备

1. 下载安装 [微信开发者工具](https://developers.weixin.qq.com/miniprogram/dev/devtools/download.html)
2. 注册微信小程序账号，获取 AppID
3. 开通微信云开发

### 本地运行

```bash
# 1. 克隆项目
git clone https://github.com/BearMaiMai/flora.git

# 2. 用微信开发者工具打开项目目录

# 3. 在开发者工具中开通云开发环境

# 4. 上传并部署云函数
```
