# 2026-04-30 工作日志

## 生成六个数据库集合 JSON 初始化文件

完成了校园快递代取/花卉小程序项目的全套数据库初始化文件：

- `flowers.json`：50 种花卉完整数据（含中文养护信息、花语、标签，参考 Oxford 102 Flowers Dataset 分类）
- `daily_tips.json`：30 条每日养护贴士（按季节/类别分类）
- `users.json`：2 条用户测试样例
- `my_plants.json`：4 条用户植物实例测试样例
- `grow_diaries.json`：6 条成长日记测试样例
- `reminders.json`：6 条养护提醒测试样例

所有文件均为标准 JSON 数组，字段引用关系：`my_plants.flowerId → flowers._id`，`grow_diaries.plantId → my_plants._id`，`reminders.plantId → my_plants._id`

**图片说明**：Oxford 102 Flowers 官网不提供直链可下载的单图，图片资源需通过其 `.tgz` 压缩包批量下载，无法在线逐一抓取。建议用户手动下载后上传至云存储并更新 `coverImage` 字段。

## 六个 JSON 文件转 JSONL 格式

微信云开发导入要求 JSON Lines 格式。已将六个文件从标准 JSON 数组（`[{...},{...}]`）转为 JSONL（每行一个独立 JSON 对象，无外层 `[]`）。实际文件名为 `plants.json`（非 `my_plants.json`）和 `diaries.json`（非 `grow_diaries.json`）。
